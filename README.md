# 띵호와

띵호와는 마인크래프트 띵타이쿤 요리 계산을 돕는 비공식 팬 사이트입니다. 는 ㄴ ㅏ혼자 쓸거지롱~

## 주요 기능

- 요리 판매가 기준 수익 계산
- 보유 재료 기준 요리 재배 계획
- 씨앗 채집 / 심기 추천
- 개인 실측 보정
- 공유 실측 보정
- 데이터 백업 / 복원

## 개인정보 안내

닉네임, 이메일, 구글 계정 등 개인 식별 정보는 전송하지 않습니다.

실측 데이터 공유 시 계산 정확도 개선을 위한 수치 데이터만 전송됩니다.

## 안내

이 사이트는 비공식 팬 사이트이며, 띵타이쿤 공식 서비스가 아닙니다.
